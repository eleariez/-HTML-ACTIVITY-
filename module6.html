Module 6: Formatting Text with CSS ✨

Introduction to Text Formatting

CSS allows you to change how text looks on a webpage. You can modify size, color, spacing, alignment, and decoration.

1. Text Color

p {
   color: hotpink;
}

2. Font Size

h1 {
   font-size: 40px;
}

3. Font Family

body {
   font-family: Arial, sans-serif;
}

4. Text Alignment

p {
   text-align: center;
}

5. Text Decoration

a {
   text-decoration: none;
}

6. Font Weight & Style

p {
   font-weight: bold;
   font-style: italic;
}

Activity

Style a paragraph with: - Your favorite color - Font size 25px - Center alignment - Bold text
