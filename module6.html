Introduction to Text Formatting

CSS allows you to modify the appearance of text on your webpage. You can control colors, fonts, sizes, spacing, alignment, decoration, and even style text on hover for interactivity.

1. Text Color

p { color: hotpink; }

2. Font Size

h1 { font-size: 40px; }

3. Font Family

body { font-family: Arial, sans-serif; }

4. Text Alignment

p { text-align: center; }

5. Text Decoration

a { text-decoration: none; }

6. Font Weight & Style

p { font-weight: bold; font-style: italic; }
