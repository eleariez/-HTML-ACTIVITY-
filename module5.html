Introduction to CSS Selectors

CSS selectors are used to target HTML elements that you want to style. Without selectors, CSS would not know which element to design.

1. Element Selector

Selects all elements of the same type.

p {
   color: hotpink;
}

2. Class Selector

A class selector starts with a dot (.) and can be used on multiple elements.

.pinkText {
   color: deeppink;
}

This text uses a class selector.

3. ID Selector

An ID selector starts with # and is unique (used only once).

#mainTitle {
   font-size: 40px;
}
