Introduction to Layout

Layout controls how elements are arranged on a webpage. Good layout makes your website clean and organized.

1. Margin

div {
   margin: 20px;
}

Margin creates space outside the element.

2. Padding

div {
   padding: 20px;
}

Padding creates space inside the element.

3. Display Property

  • block
  • inline
  • inline-block
  • flex

4. Flexbox (Side by Side Layout)

.container {
   display: flex;
}

Flexbox allows elements to appear side by side.
